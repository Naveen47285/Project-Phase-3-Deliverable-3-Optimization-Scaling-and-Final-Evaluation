# Phase 3: Optimization, Scaling, and Final Evaluation

This project implements an optimized Min Heap data structure in Python as part of Phase 3 of the assignment.

## 🔧 Features
- Iterative heapify (optimized)
- Insert and extract operations
- Bulk heap construction (build_heap)
- Performance tracking using operation count
- Scalable testing with large datasets
- Stress testing (20,000 elements)
- Graph-based performance visualization

## 📁 Files
- heap_optimized.py → Optimized Min Heap implementation
- main.py → Basic demonstration
- test_heap.py → Unit testing
- performance_test.py → Performance and stress testing
- graph.py → Visualization of performance scaling
- Phase3_Optimization_MinHeap.pdf → Final report

## 📊 Performance
Tested with dataset sizes:
- 100
- 1,000
- 5,000
- 10,000
- Stress test with 20,000 elements
